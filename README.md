# may-database

MAY platformu PostgreSQL veritabanı.

## Teknolojiler

- **PostgreSQL 17**
- **Docker Compose**

## Kullanım

```bash
# Başlat
docker compose up -d

# Durdur
docker compose down

# Logları gör
docker compose logs -f
```

## Bağlantı Bilgileri

| Parametre | Değer |
|---|---|
| Host | localhost |
| Port | 5432 |
| User | postgres |
| Password | 123456789 |
